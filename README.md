# Shopping List - Список покупок 🚀🎉⚡️

Удобное приложение для составления и управления списками покупок.

## 🌟 Overview

**Shopping List** - это прогрессивное веб-приложение (PWA) для создания списков покупок, управления их содержимым и совместного доступа.

## ✨ Tech Stack & Features

- **Frontend**: React 19, Vite 6, TypeScript, MUI 6
- **Routing**: React Router v7
- **State Management**: Jotai
- **Authentication**: Keycloak (OIDC) via Backend for Frontend (BFF)
- **PWA Support**: Работает оффлайн и устанавливается на любое устройство
- **Internationalization**: i18next (Русский/English)

## 🚀 Getting Started

### Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

### Available Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run prettier:check` | Check formatting |
| `npm run lint:check` | Check linting |
| `npm run ts:check` | Check TypeScript |
| `npm run test:unit` | Run unit tests |
| `npm run test:e2e` | Run e2e tests |
